Create Event
Steps:

1. Click on the “Add New Event” button

2. Enter event information

3. Please ensure the starting time is after the ending time otherwise an error alert will show

4. Click on the "Create Event" button

5. Read and confirm the entered information, and click on the "Confirm" button


Search Event
Steps:

1. Click on the search input field

2. Enter a search criteria


Duplicate Event
Steps:

1. Slide the scheduled event you want to duplicate to your left

2. Click on the “Duplicate” button

3. Change the event information

4. Please ensure the starting time is after the ending time otherwise an error alert will show

5. Please ensure the duplication is unique otherwise an alert will show

6. For the duplication to be unique, please change atleast one of the following:

a. The venue for the event

b. The employee for the event

c. The date range may not overlap with the duplicated event if neither the venue or employee is changed

7. Click on the "Duplicate Event" button

8. Read and confirm the newly entered information, and click on the "Confirm" button


Delete Event
Steps:

1. Slide the event you want to delete to your left

2. Click on the “Delete” button

3. Confirm deletion by clicking on the "Delete Event" button

4. If there are client bookings currently associated to the booking type, it will not be able to delete. The associated client bookings need to be cancelled first.
