Generate Income Report
Steps:

1. The income report contains information regarding the profit generated per each sale category at BSC gym

2. The default date aggregation is set to 'monthly'. If you wish to view a different aggregation click on the "select chart view" drop down and select which aggregation you wish to see

3. Download the report as a PDF by clicking on the download button

4. If the report does not download, refresh the page and try again
